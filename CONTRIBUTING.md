# Contributing to Resume Updates

Thank you for your interest in contributing to this resume repository!

## How to Contribute

### Reporting Issues

If you find any issues with the LaTeX template or build process:
1. Check if the issue already exists
2. Create a new issue with a clear description
3. Include steps to reproduce if applicable

### Making Changes

1. Fork the repository
2. Create a new branch for your changes
3. Make your modifications
4. Test the compilation with `make` or `pdflatex base_resume.tex`
5. Submit a pull request

### LaTeX Best Practices

- Keep the code clean and well-commented
- Follow the existing formatting style
- Test compilation before submitting changes
- Ensure the PDF output looks professional

### Customizing for Different Applications

When creating tailored versions:
1. Copy `base_resume.tex` to a new file
2. Modify content as needed
3. Save in the appropriate output directory
4. Keep the base template generic

## Questions?

Feel free to open an issue for any questions or suggestions!
